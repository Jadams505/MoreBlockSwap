Allows for more tiles to be block swapped with each other. Works for vanilla and modded tiles.

[h1]General Rules For Swapping[/h1]
[list]
[*]Tiles must either be the same type or the same size
[*]Tiles that place tile entities can not be swapped to or from this includes tiles like: chests, item frames, plates, etc
[*]Cannot swap a non solid tile to a solid tile if a player is in the way
[*]The player must have enough pick power to mine the block
[/list]

[h1]Examples[/h1]
[list]
[*]Bookcases
[*]Beds
[*]Banners
[*]Paintings
[*]Clocks
[*]Tombstones
[*]Relics
[*]Grasses
[*]Moss
[*]Signs
[*]Cages
[*]Doors (open and closed)
[*]And Many More
[/list]
